1. **按顺序学**：从第0章开始，不要跳章，每章包含前置知识提示
2. **动手敲代码**：每段示例代码都要亲手运行一遍，光看记不住
3. **先做再看**：每章末尾的「典型考题」先自己写，再展开参考答案
4. **错题标记**：做错的题标记出来，Day 5 重点回顾
5. **考前30分钟**：只看 [考前速记卡](./appendix/quick-review.md) 和错题
