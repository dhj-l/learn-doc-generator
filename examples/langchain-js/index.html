📌 学习建议

🎯 高效学习三步法

  1. 先读概念 — 每个知识点先理解「是什么」和「为什么需要」
  2. 动手敲代码 — 每个代码示例都亲手敲一遍，不要复制粘贴
  3. 完成练习 — 每章末尾的实战练习是巩固知识的关键
💡 你将学到什么
完成本教程后，你将能够：
  • 使用 LangChain.js 构建功能完整的 AI Agent
  • 连接 OpenAI、Anthropic、Google 等主流模型提供商
  • 为 Agent 创建自定义工具，让它具备查询数据库、调用 API 等能力
  • 实现流式响应、结构化输出等高级特性
  • 通过中间件为 Agent 添加日志、防护栏等企业级功能
  • 构建一个完整的智能客服助手项目
  • 掌握提示模板和 LCEL 声明式链式组合
  • 实现 RAG（检索增强生成）让 AI 拥有专业知识库
  • 使用 LangGraph 编排复杂多步骤工作流
  • 通过 Callbacks 和 LangSmith 追踪调试 Agent
