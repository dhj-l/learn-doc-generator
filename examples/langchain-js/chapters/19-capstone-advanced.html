🚀 进阶实战：RAG + LangGraph 智能文档助手

融合 RAG 检索增强生成与 LangGraph 工作流编排，构建企业级智能文档问答系统

🎯 本章目标

这是本教程的最终章——一个面向真实场景的 综合进阶实战项目。在前十章基础实战和后续 RAG 专题（第13–16章）、LangGraph（第17章）、Callbacks（第18章）的学习基础上，我们将把 文档加载与分割、Embedding 向量化、向量存储与检索、RAG 检索增强生成、LangGraph 状态图编排、流式传输、记忆系统 以及 LangSmith 可观测性 全部串联起来，构建一个真正的智能文档助手（Intelligent Document Assistant）。

这个项目覆盖了企业级 RAG 应用开发的完整链路：从文档上传 → 索引构建 → 语义检索 → 多轮对话 → 工作流编排 → 生产监控。你将学习如何用 LangGraph 编排条件路由的工作流，用 MemoryVectorStore 实现轻量级向量检索，并通过 LangSmith 实现端到端链路追踪。

💡 学习建议

这是综合性最强的章节，建议先理解整体架构，再逐步阅读各步骤的代码实现。如果对 RAG 或 LangGraph 的基础概念不熟悉，可以回顾第13–17章的专题内容。

📋 项目概述

我们要构建的是一个 智能文档助手（Intelligent Document Assistant），它能够像一位专业的文档顾问一样，理解用户上传的文档内容，回答关于文档的各种问题，进行摘要总结，并在多轮对话中保持上下文记忆。

核心功能清单

  • 📄 文档上传与索引 — 用户上传文档后，系统自动完成加载、文本分割、向量化并存入向量库
  • 🔍 智能问答（RAG） — 用户提问时，系统从文档中检索相关内容，结合大模型生成精准答案
  • 📝 文档摘要 — 自动生成文档的概要总结，支持不同粒度的摘要（段落级/全文级）
  • 💬 多轮对话 — 借助记忆系统，在同一会话中保持上下文，支持追问和澄清
  • 🔄 条件路由 — 根据用户意图自动分派到不同的处理流程（问答、摘要、闲聊等）
  • ⚡ 流式输出 — 所有生成内容以流式方式实时推送，提升交互体验
  • 📊 可观测追踪 — 通过 LangSmith 记录每次查询的完整链路，方便调试和优化
🔧 技术栈概览

本项目使用 LangChain.js v1.x + LangGraph + OpenAI Embeddings + MemoryVectorStore 构建，TypeScript 编写，运行在 Node.js 18+ 环境。所有向量数据存储在内存中，方便快速原型验证，生产环境可无缝切换到 Chroma/Pinecone 等持久化向量库。

📦 Step 1：文档索引管线

文档索引管线 — rag/pipeline.ts

这是索引管线的核心实现：

src/rag/pipeline.ts 

import { TextLoader } from "langchain/document_loaders/fs/text";
import { RecursiveCharacterTextSplitter } from "langchain/text_splitter";
import { MemoryVectorStore } from "langchain/vectorstores/memory";
import { Document } from "@langchain/core/documents";
import { initEmbeddings } from "../config.js";
import { config } from "../config.js";

/**
 * 步骤一：加载文档
 * 支持 .txt / .md / .json / .csv 格式
 */
export async function loadDocument(filePath: string): Promise<Document[]> {
  const loader = new TextLoader(filePath);
  const docs = await loader.load();

  // 为文档添加元数据
  const fileName = filePath.split("/").pop() || "unknown";
  return docs.map((doc) => ({
    ...doc,
    metadata: {
      ...doc.metadata,
      source: fileName,
      loadedAt: new Date().toISOString(),
    },
  }));
}

/**
 * 步骤二：文本分割
 * 使用 RecursiveCharacterTextSplitter 按段落/句子递归分割
 */
export async function splitDocuments(docs: Document[]): Promise<Document[]> {
  const splitter = new RecursiveCharacterTextSplitter({
    chunkSize: config.chunkSize,
    chunkOverlap: config.chunkOverlap,
    separators: ["\n\n", "\n", "。", ".", " ", ""],
  });

  const splitDocs = await splitter.splitDocuments(docs);

  // 为每个分块添加序号
  return splitDocs.map((doc, index) => ({
    ...doc,
    metadata: {
      ...doc.metadata,
      chunkIndex: index,
      totalChunks: splitDocs.length,
    },
  }));
}

/**
 * 步骤三 + 四：向量化并存入 MemoryVectorStore
 */
export async function embedAndStore(
  docs: Document[]
): Promise<MemoryVectorStore> {
  const embeddings = initEmbeddings();
  const vectorStore = new MemoryVectorStore(embeddings);

  // 批量添加文档（内部自动完成向量化）
  await vectorStore.addDocuments(docs);

  console.log(
    `✅ 索引完成: ${docs.length} 个文档片段已向量化并存储`
  );

  return vectorStore;
}

/**
 * 一站式管线：加载 → 分割 → 向量化 → 存储
 */
export async function buildIndex(filePath: string): Promise<MemoryVectorStore> {
  console.log(`📂 加载文档: ${filePath}`);
  const rawDocs = await loadDocument(filePath);
  console.log(`   ├─ 原始文档数: ${rawDocs.length}`);

  const splitDocs = await splitDocuments(rawDocs);
  console.log(`   ├─ 分割后片段数: ${splitDocs.length}`);

  const vectorStore = await embedAndStore(splitDocs);
  console.log(`   └─ ✅ 向量存储就绪`);

  return vectorStore;
}
🔍 索引管线关键设计
  • 递归分割 — 使用 RecursiveCharacterTextSplitter 按中英文标点和换行符分层分割，保证语义完整性
  • 元数据追踪 — 每个分块记录来源文件名、块索引、总块数，方便溯源
  • 批量向量化 — addDocuments 内部自动批量调用 Embedding API，效率更高
  • 内存存储 — MemoryVectorStore 零配置，生产环境可替换为 ChromaVectorStore 或 PineconeVectorStore

检索器封装 — rag/retriever.ts

src/rag/retriever.ts 

import { MemoryVectorStore } from "langchain/vectorstores/memory";
import { Document } from "@langchain/core/documents";
import { config } from "../config.js";

/**
 * 创建检索器
 * 封装相似度检索逻辑，支持配置返回数量
 */
export function createRetriever(vectorStore: MemoryVectorStore) {
  return vectorStore.asRetriever(config.retrieverK, {
    // 可按元数据过滤（例如按文档来源）
    // filter: (doc) => doc.metadata.source === "xxx.md",
  });
}

/**
 * 执行检索并格式化结果
 * 将 Document 列表转为可供 LLM 使用的上下文文本
 */
export async function retrieveContext(
  vectorStore: MemoryVectorStore,
  query: string
): Promise<{ documents: Document[]; context: string }> {
  const retriever = createRetriever(vectorStore);
  const documents = await retriever.invoke(query);

  // 将检索结果格式化为带来源标记的上下文文本
  const context = documents
    .map((doc, i) => {
      const source = doc.metadata.source || "未知来源";
      const chunkInfo = doc.metadata.chunkIndex !== undefined
        ? `[片段 ${doc.metadata.chunkIndex + 1}/${doc.metadata.totalChunks}]`
        : "";
      return `【来源 ${i + 1}】${source} ${chunkInfo}\n${doc.pageContent}`;
    })
    .join("\n\n---\n\n");

  return { documents, context };
}

/**
 * 格式化检索结果为引用列表（用于显示来源）
 */
export function formatSources(documents: Document[]): string {
  const seen = new Set<string>();
  const sources: string[] = [];

  for (const doc of documents) {
    const source = doc.metadata.source as string;
    const chunkIdx = doc.metadata.chunkIndex as number;
    const key = `${source}#${chunkIdx}`;

    if (!seen.has(key)) {
      seen.add(key);
      sources.push(`📄 ${source} (片段 ${(chunkIdx ?? 0) + 1})`);
    }
  }

  return sources.length > 0
    ? `\n\n📚 参考来源：\n${sources.join("\n")}`
    : "";
}

/**
 * 节点 1：意图分类
 * 使用 LLM 判断用户意图
 */
export async function classifyNode(state: GraphStateType) {
  const llm = initLLM();
  const prompt = ChatPromptTemplate.fromMessages([
    ["system", CLASSIFY_SYSTEM_PROMPT],
    ["human", "{input}"],
  ]);

  const chain = prompt.pipe(llm).pipe(new StringOutputParser());
  const result = await chain.invoke({ input: state.input });

  // 解析意图
  const intent = parseIntent(result.trim().toLowerCase());
  console.log(`🔍 [意图识别] ${state.input} → ${intent}`);

  return {
    ...state,
    intent,
    messages: [
      ...state.messages,
      new HumanMessage(state.input),
    ],
  };
}

/**
 * 节点 2：RAG 问答
 * 检索文档并生成基于上下文的回答
 */
export async function qaNode(state: GraphStateType) {
  if (!_vectorStore) {
    throw new Error("向量存储未初始化，请先调用 buildIndex()");
  }

  // 步骤一：检索相关上下文
  const { documents, context } = await retrieveContext(
    _vectorStore,
    state.input
  );

  // 步骤二：生成回答
  const llm = initLLM();
  const prompt = ChatPromptTemplate.fromMessages([
    ["system", QA_SYSTEM_PROMPT],
    ["human", "{input}"],
  ]);

  const chain = prompt.pipe(llm).pipe(new StringOutputParser());
  const answer = await chain.invoke({
    context,
    input: state.input,
  });

  // 步骤三：格式化来源引用
  const sources = formatSources(documents);

  return {
    ...state,
    context,
    retrievedDocs: documents,
    answer,
    sources,
  };
}

/**
 * 节点 3：文档摘要
 * 对文档内容进行摘要总结
 */
export async function summarizeNode(state: GraphStateType) {
  if (!_vectorStore) {
    throw new Error("向量存储未初始化，请先调用 buildIndex()");
  }

  // 检索所有文档内容用于摘要
  const allDocs = await _vectorStore.similaritySearch("", 100);
  const fullContent = allDocs.map((d: any) => d.pageContent).join("\n\n");

  const llm = initLLM();
  const prompt = ChatPromptTemplate.fromMessages([
    ["system", "你是一个专业的文档摘要助手。" + SUMMARIZE_PROMPT],
    ["human", "请根据用户需求进行摘要：{input}"],
  ]);

  const chain = prompt.pipe(llm).pipe(new StringOutputParser());
  const summary = await chain.invoke({
    content: fullContent.slice(0, 15000), // 限制 token 数
    input: state.input,
  });

  return {
    ...state,
    answer: summary,
    sources: `📚 基于 ${allDocs.length} 个文档片段的全局摘要`,
  };
}

/**
 * 节点 4：闲聊
 * 处理与文档无关的一般性对话
 */
export async function chatNode(state: GraphStateType) {
  const llm = initLLM();
  const prompt = ChatPromptTemplate.fromMessages([
    ["system", CHAT_SYSTEM_PROMPT],
    ...state.messages.slice(-6).map((m) =>
      m instanceof HumanMessage
        ? ["human", m.content] as const
        : ["ai", m.content] as const
    ),
    ["human", "{input}"],
  ]);

  const chain = prompt.pipe(llm).pipe(new StringOutputParser());
  const answer = await chain.invoke({ input: state.input });

  return {
    ...state,
    answer,
    sources: "",
  };
}

/**
 * 节点 5：输出格式化
 * 将结果整理为最终输出
 */
export async function outputNode(state: GraphStateType) {
  const finalAnswer = state.answer + (state.sources ? `\n\n${state.sources}` : "");
  const aiMessage = new AIMessage(finalAnswer);

  return {
    ...state,
    messages: [...state.messages, aiMessage],
    answer: finalAnswer,
  };
}

// ========== 辅助函数 ==========

function parseIntent(raw: string): Intent {
  if (raw.includes("summarize") || raw.includes("summary") || raw.includes("总结") || raw.includes("摘要") || raw.includes("概括")) {
    return "summarize";
  }
  if (raw.includes("qa") || raw.includes("chat") && (raw.includes("文档") || raw.includes("内容") || raw.includes("什么") || raw.includes("如何"))) {
    return "qa";
  }
  if (raw.includes("qa")) return "qa";
  if (raw.includes("chat")) return "chat";
  // 默认走 QA 路由（可调整）
  return "qa";
}

import { StateGraph, END } from "@langchain/langgraph";
import { GraphState } from "./state.js";
import {
  classifyNode,
  qaNode,
  summarizeNode,
  chatNode,
  outputNode,
} from "./nodes.js";

/**
 * 条件路由函数
 * 根据意图分类结果决定下一个执行的节点
 */
function routeIntent(state: typeof GraphState.StateType): string {
  const intent = state.intent;
  console.log(`🔄 [路由决策] 意图: ${intent}`);

  switch (intent) {
    case "qa":
      return "qa_node";
    case "summarize":
      return "summarize_node";
    case "chat":
      return "chat_node";
    default:
      return "qa_node"; // 默认走 QA
  }
}

/**
 * 构建 LangGraph 工作流
 */
export function buildWorkflow() {
  // 1. 定义图
  const workflow = new StateGraph(GraphState)
    // 2. 添加节点
    .addNode("classify_node", classifyNode)
    .addNode("qa_node", qaNode)
    .addNode("summarize_node", summarizeNode)
    .addNode("chat_node", chatNode)
    .addNode("output_node", outputNode)

    // 3. 设置入口边 — 从分类节点开始
    .addEdge("__start__", "classify_node")

    // 4. 条件路由 — 分类完成后根据意图路由
    .addConditionalEdges("classify_node", routeIntent, {
      qa_node: "qa_node",
      summarize_node: "summarize_node",
      chat_node: "chat_node",
    })

    // 5. 各处理节点完成后汇聚到输出节点
    .addEdge("qa_node", "output_node")
    .addEdge("summarize_node", "output_node")
    .addEdge("chat_node", "output_node")

    // 6. 输出节点后结束
    .addEdge("output_node", END);

  // 7. 编译图
  return workflow.compile();
}

/**
 * 运行工作流
 */
export async function runWorkflow(input: string) {
  const app = buildWorkflow();

  const initialState = {
    messages: [],
    input,
    intent: "qa" as const,
    context: "",
    retrievedDocs: [],
    answer: "",
    sources: "",
    error: null,
  };

  const finalState = await app.invoke(initialState);
  return finalState;
}
⚠️ 条件路由关键概念

🧠 Step 4：记忆系统集成

为了让智能文档助手支持多轮对话，我们需要集成记忆系统。LangGraph 内置了 MemorySaver 持久化机制，结合消息历史管理，可以实现会话级别的上下文记忆。

记忆管理 — memory/history.ts

src/memory/history.ts 

import { BaseMessage, HumanMessage, AIMessage } from "@langchain/core/messages";
import { ChatMessageHistory } from "langchain/stores/message/in_memory";

/**
 * 会话记忆管理器
 * 封装 ChatMessageHistory，提供便捷的读写接口
 */
export class ConversationMemory {
  private history: ChatMessageHistory;
  private maxHistoryPairs: number;

  constructor(maxHistoryPairs = 10) {
    this.history = new ChatMessageHistory();
    this.maxHistoryPairs = maxHistoryPairs;
  }

  /** 添加用户消息 */
  async addUserMessage(content: string): Promise<void> {
    await this.history.addMessage(new HumanMessage(content));
  }

  /** 添加 AI 消息 */
  async addAIMessage(content: string): Promise<void> {
    await this.history.addMessage(new AIMessage(content));
  }

  /** 获取最近的消息列表（自动截断） */
  async getRecentMessages(): Promise<BaseMessage[]> {
    const messages = await this.history.getMessages();
    // 保留最近 N 轮对话（N * 2 条消息）
    if (messages.length > this.maxHistoryPairs * 2) {
      return messages.slice(-this.maxHistoryPairs * 2);
    }
    return messages;
  }

  /** 格式化历史消息为文本（用于嵌入 prompt） */
  async formatHistory(): Promise<string> {
    const messages = await this.getRecentMessages();
    return messages
      .map((msg) => {
        const role = msg instanceof HumanMessage ? "用户" : "助手";
        const content =
          typeof msg.content === "string"
            ? msg.content
            : JSON.stringify(msg.content);
        return `${role}: ${content}`;
      })
      .join("\n");
  }

  /** 清空历史 */
  async clear(): Promise<void> {
    await this.history.clear();
  }
}

/**
 * 将记忆集成到 LangGraph 工作流中
 * 通过 checkpointer 实现状态持久化
 */
import { MemorySaver } from "@langchain/langgraph";

export function createCheckpointer() {
  return new MemorySaver();
}

⚡ Step 5：流式输出

流式输出可以显著提升用户体验。LangGraph 和 LangChain 都提供了流式事件（streamEvents）机制，我们可以逐 token 地推送生成结果。

流式处理器 — streaming/handler.ts

src/streaming/handler.ts 

import { StateGraph } from "@langchain/langgraph";
import { GraphState } from "../graph/state.js";
import {
  classifyNode,
  qaNode,
  summarizeNode,
  chatNode,
  outputNode,
} from "../graph/nodes.js";
import { routeIntent } from "../graph/workflow.js"; // 注意：需导出 routeIntent

/**
 * 构建可流式的工作流
 * 返回编译后的图，支持 .stream() 调用
 */
export function buildStreamableWorkflow() {
  const workflow = new StateGraph(GraphState)
    .addNode("classify_node", classifyNode)
    .addNode("qa_node", qaNode)
    .addNode("summarize_node", summarizeNode)
    .addNode("chat_node", chatNode)
    .addNode("output_node", outputNode)
    .addEdge("__start__", "classify_node")
    .addConditionalEdges("classify_node", routeIntent, {
      qa_node: "qa_node",
      summarize_node: "summarize_node",
      chat_node: "chat_node",
    })
    .addEdge("qa_node", "output_node")
    .addEdge("summarize_node", "output_node")
    .addEdge("chat_node", "output_node")
    .addEdge("output_node", END);

  return workflow.compile();
}

/**
 * 流式运行工作流
 * 逐 token 输出生成结果
 */
export async function* streamWorkflow(input: string) {
  const app = buildStreamableWorkflow();

  const initialState = {
    messages: [],
    input,
    intent: "qa" as const,
    context: "",
    retrievedDocs: [],
    answer: "",
    sources: "",
    error: null,
  };

  // 使用 streamEvents 获取细粒度的事件流
  const eventStream = await app.streamEvents(initialState, {
    version: "v1",
  });

  for await (const event of eventStream) {
    // 只输出 LLM 生成的 token 事件
    if (event.event === "on_chat_model_stream") {
      const token = event.data?.chunk?.content;
      if (token) {
        yield {
          type: "token",
          content: typeof token === "string" ? token : JSON.stringify(token),
        };
      }
    }

    // 输出节点完成事件（用于显示进度）
    if (event.event === "on_chain_end" && event.name?.endsWith("_node")) {
      yield {
        type: "node_complete",
        node: event.name,
      };
    }
  }

  // 最终状态
  yield {
    type: "done",
  };
}

/**
 * CLI 流式输出演示
 */
export async function streamChat(input: string) {
  process.stdout.write("\n🤖 [助手] ");
  let fullAnswer = "";

  for await (const chunk of streamWorkflow(input)) {
    if (chunk.type === "token") {
      process.stdout.write(chunk.content);
      fullAnswer += chunk.content;
    } else if (chunk.type === "node_complete") {
      // 节点完成时不输出到终端，仅内部追踪
    } else if (chunk.type === "done") {
      process.stdout.write("\n\n");
    }
  }

  return fullAnswer;
}

import { Run } from "@langchain/core/tracers/base";

/**
 * 自定义追踪器
 * 继承 LangChainTracer 并扩展日志记录功能
 */
export class CustomTracer extends LangChainTracer {
  private startTime: number = 0;

  constructor(fields?: Partial<LangChainTracerFields>) {
    super(fields);
  }

  /** 记录 LLM 调用开始 */
  async onLLMStart(run: Run): Promise<void> {
    this.startTime = Date.now();
    console.log(`📡 [追踪] LLM 调用开始: ${run.name}`);
    await super.onLLMStart(run);
  }

  /** 记录 LLM 调用结束 */
  async onLLMEnd(run: Run): Promise<void> {
    const duration = Date.now() - this.startTime;
    console.log(`📡 [追踪] LLM 调用完成, 耗时: ${duration}ms`);
    await super.onLLMEnd(run);
  }

  /** 记录工具调用 */
  async onToolStart(run: Run): Promise<void> {
    console.log(`📡 [追踪] 工具调用: ${run.name}`);
    console.log(`   └─ 输入: ${JSON.stringify(run.inputs).slice(0, 200)}`);
    await super.onToolStart(run);
  }

  /** 记录检索操作 */
  async onRetrieverStart(run: Run): Promise<void> {
    console.log(`📡 [追踪] 检索开始: "${JSON.stringify(run.inputs)}"`);
    await super.onRetrieverStart(run);
  }

  async onRetrieverEnd(run: Run): Promise<void> {
    const docsCount = (run.outputs as any)?.documents?.length ?? 0;
    console.log(`📡 [追踪] 检索完成, 返回 ${docsCount} 个文档`);
    await super.onRetrieverEnd(run);
  }
}

/**
 * 初始化 LangSmith 追踪
 * 如果环境变量已配置，LangChain 会自动启用追踪，
 * 此函数仅用于确认配置状态
 */
export function initTracing() {
  const isEnabled = process.env.LANGSMITH_TRACING === "true";
  const project = process.env.LANGSMITH_PROJECT || "default";

  if (isEnabled) {
    console.log(`📊 LangSmith 追踪已启用, 项目: ${project}`);
    console.log(`   🔗 https://smith.langchain.com/studio?project=${project}`);
  } else {
    console.log("📊 LangSmith 追踪未启用（设置 LANGSMITH_TRACING=true 开启）");
  }

  return isEnabled;
}
