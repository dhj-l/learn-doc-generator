## 💡 核心概念

### 18.1 项目概述

**用一个类比来理解：**

> 想象你是一个杂志社的内容总监。你手下有几位专门的内容创作者：一位擅长写科技文章，一位擅长写营销文案，还有一位擅长配图设计。你只需要告诉它们主题和风格要求，它们就能产出符合标准的成品内容。你不需要自己动手写每一个字，只需要管理好创作流程。这就是内容创作 Agent 的工作方式——你提供主题和要求，它自动完成所有创作工作。
>
> 本章的实战项目将构建一个内容创作 Agent，它能够根据用户提供的主题和风格要求，自动生成完整的文章内容。它内置了文章生成和配图描述两个核心能力，可以产出图文并茂的完整内容，适用于博客写作、营销文案和新闻稿等多种场景。

**核心能力说明：**

内容创作 Agent 的核心是两个工具，分别对应内容创作流程中的关键步骤。文章生成工具负责内容的创作和撰写，配图生成工具负责为内容添加合适的视觉元素，两者配合可以产出图文并茂的完整内容，满足博客、新闻稿、营销文案等多种场景的需求。

1. **write_article（文章生成工具）** —— 接收用户指定的主题和风格参数，生成一篇结构完整的文章正文内容。支持四种不同的写作风格：正式风格适合新闻稿和官方公告等正式场合的写作需求，轻松风格适合博客文章和社交媒体等日常内容创作场景，技术风格适合技术教程和开发文档等技术性较强的内容，营销风格适合产品推广和广告文案等营销类内容的创作场景，覆盖了不同类型的内容需求场景。

2. **generate_image（配图生成工具）** —— 根据已生成的文章内容自动创建配图描述方案，包括建议的图片类型、视觉风格选择和需要突出的内容要点信息说明。通过为文章添加合适的视觉元素，可以显著提升内容的阅读体验和传播效果，让文章在视觉上更有吸引力和说服力，帮助内容在众多信息中脱颖而出并吸引读者关注。

这两个工具配合使用，可以完成从用户提供的主题到最终成品的完整内容创作工作流程步骤。用户只需要提供主题和风格偏好，Agent 会自动完成从文章撰写到配图生成的全部工作内容。Agent 的 LLM 会根据用户需求自动判断调用哪个工具以及调用顺序，无需人工干预处理。这种灵活的工具编排能力使得 Agent 可以适应不同类型的内容创作需求，无论是写博客文章还是生成营销文案都能应对自如。

### 18.3 系统提示设计

内容创作 Agent 的系统提示包含了风格指南，分别定义了四种写作风格的具体特征和适用场景要求说明内容。这比让 Agent 自由发挥要可靠得多——明确的风格定义确保了输出的一致性和可预测性，让用户每次获得的内容都符合预期的风格标准和内容质量要求标准。系统提示中还定义了清晰的工作流程步骤，引导 Agent 按正确的顺序调用工具，先写文章正文再生成配图描述方案，避免出现流程混乱或步骤遗漏的问题。

### 18.4 代码说明

上述代码实现了内容创作 Agent 的两个核心工具。write_article 根据主题和风格生成文章正文，generate_image 为文章创建配图方案。Agent 先分析用户需求，然后依次调用两个工具完成任务。在实际生产环境中，这些模拟工具可以替换为真实的 API 调用，接入 GPT 或 Claude 来生成高质量的文章内容。

### 18.5 系统提示和流程设计

内容创作 Agent 的系统提示定义了工作流程和风格指南。系统提示中明确了先写文章再配图的步骤顺序，以及四种写作风格的具体特征和要求。风格指南确保了输出内容的一致性和可预测性，让用户每次获得的内容都符合预期的风格标准。

在实际生产环境中，write_article 工具可以接入 GPT 或 Claude 等大语言模型来生成高质量的文章内容。generate_image 工具可以接入 DALL-E 或 Stable Diffusion 等图像生成模型来创建真实的配图。将模拟工具替换为真实 API 后，Agent 就可以直接生成可用于实际发布的优质内容了。

### 18.2 核心代码实现

```typescript
import { createDeepAgent } from "deepagents";
import { tool } from "langchain";
import { z } from "zod";

// 文章生成工具，根据用户指定的主题和风格参数生成一篇结构完整的文章正文内容
// 支持四种不同的写作风格，分别适用于不同的内容创作场景和实际需求：
// 正式风格适合新闻稿和官方公告等正式场合的写作需求场景和内容类型
// 轻松风格适合博客文章和社交媒体等日常内容创作场景和需求
// 技术风格适合技术教程和开发文档等技术性较强的内容创作场景
// 营销风格适合产品推广和广告文案等营销类内容的创作场景和需求
const writeArticle = tool(
  async ({ topic, tone }: { topic: string; tone: string }) => {
    return `# ${topic}\n\n## 引言\n这是一篇关于${topic}的${tone}风格文章。\n\n## 正文\n本文将从多个角度探讨${topic}的相关话题...\n\n## 总结\n通过以上分析，我们可以得出...`;
  },
  {
    name: "write_article",
    description: "根据主题和风格生成一篇完整的文章",
    schema: z.object({
      topic: z.string().describe("文章主题，如量子计算入门"),
      tone: z.enum(["正式", "轻松", "技术", "营销"]).describe("文章风格"),
    }),
  }
);

// 图片描述生成工具，为已生成的文章内容自动创建配图方案和建议内容
// 根据文章的主题和核心内容自动建议合适的图片类型和视觉风格
// 帮助实现图文并茂的内容输出效果，提升文章可读性和传播效果
const generateImageDesc = tool(
  async ({ article }: { article: string }) => {
    return `配图建议：为文章生成一张信息图，风格与内容主题一致。`;
  },
  {
    name: "generate_image",
    description: "根据文章内容生成配图描述",
    schema: z.object({
      article: z.string().describe("文章内容"),
    }),
  }
);

const agent = createDeepAgent({
  model: "anthropic:claude-sonnet-4-6",
  tools: [writeArticle, generateImageDesc],
  systemPrompt: `你是一个内容创作助手。

## 工作流程
1. 根据用户提供的主题和风格要求生成文章正文
2. 为生成的文章创建配图描述方案
3. 输出完整的文章内容和配图建议

## 风格指南
- 正式风格：专业术语，结构严谨，适合新闻稿
- 轻松风格：通俗易懂，适合博客文章
- 技术风格：技术细节和代码示例，适合教程
- 营销风格：强调价值和卖点，适合推广`,
});

async function main() {
  // 用户提出内容创作需求，Agent 会根据需求自动选择合适的工具
  // 先调用 write_article 生成文章正文
  // 再调用 generate_image 为文章创建配图描述方案
  const result = await agent.invoke({
    messages: [
      { role: "user", content: "写一篇关于量子计算入门的技术博客文章，风格要轻松易懂" },
    ],
  });
  // 输出最终生成的完整内容，包括文章正文和配图建议
  console.log(result.messages.at(-1)?.content);
}
main().catch(console.error);
```

### 18.3 代码说明

上述代码定义了内容创作 Agent 的两个核心工具模块功能。write_article 工具负责根据用户提供的主题和风格参数生成结构完整的文章正文内容，支持四种不同风格以适应各种内容创作场景和需求。generate_image 工具负责为已生成的文章内容创建配图描述方案，包括建议的图片类型选择、视觉风格选择和需要突出的内容要点信息说明。Agent 在运行时先调用 write_article 生成文章正文，再调用 generate_image 创建配图方案，最后输出完整的图文内容方案给用户查看和使用。

这些工具可以接入真实的 AI 服务来提升内容的质量和实际效果。例如，write_article 可以调用 GPT 或 Claude 等大语言模型来生成高质量的文章内容，generate_image 可以调用 DALL-E 或 Stable Diffusion 等流行的图像生成模型来创建真实的配图。此外，还可以通过添加更多工具来进一步扩展 Agent 的能力范围：

- **翻译工具**：实现文章的多语言自动翻译功能，覆盖不同国家和语言的读者群体和目标市场区域
- **校对工具**：实现自动语法检查和文字校对功能，确保文章的专业性、流畅性和准确性
- **SEO 分析工具**：生成关键词优化建议和搜索引擎优化策略说明，帮助内容在搜索引擎中获得更好的排名和曝光机会
- **内容摘要工具**：自动生成文章摘要和核心要点提炼说明，方便读者快速了解文章的主要内容和核心信息

## 🚀 项目应用场景

内容创作 Agent 可以应用于多种实际场景：

- **博客写作**：自动生成技术博客、行业分析文章和个人随笔等内容创作任务和需求场景，有效提高内容产出效率。只需提供主题和风格，Agent 即可在几秒内生成初稿供你修改完善和最终发布
- **营销文案**：快速生成产品介绍、广告文案和推广邮件等内容素材，支持 A/B 测试不同风格的效果，帮助营销团队提升内容产出效率和质量
- **多语言内容**：结合翻译工具扩展，将内容输出到不同语言的市场和地区，覆盖更多读者受众群体
- **社交媒体**：为不同平台（微博、公众号、LinkedIn）生成适配的内容格式和长度要求，实现多渠道内容分发和覆盖

## 💡 使用提示

为了让内容创作 Agent 发挥最佳效果，建议注意以下几点：

1. **明确风格要求**：在请求中指定风格（正式/轻松/技术/营销），让输出更符合你的预期
2. **提供参考示例**：如果你有偏好的文章风格，可以给 Agent 提供一篇参考文章作为示例
3. **多轮迭代**：先让 Agent 生成初稿，然后提出修改意见，逐步完善内容质量

## ⚡ 进阶技巧

### 技巧一：内容模板管理

定义多套内容模板（新闻稿格式、博客格式、营销文案格式），让 Agent 根据不同类型的需求选择最合适的模板来生成内容。这样可以确保输出内容的结构一致性和质量水平，避免每次生成的格式都不一样。同时，模板也减少了 Agent 在结构设计上的 Token 消耗，让它可以更专注于内容本身的质量和深度。建议根据你的业务场景预先定义好 3-5 套常用的内容模板。

### 技巧二：多轮迭代优化

让 Agent 先生成初稿，然后根据用户的反馈进行多轮修改完善，直到用户满意为止。可以在系统提示中要求 Agent 在生成初稿后主动询问用户是否有需要调整的地方，然后根据用户提出的反馈意见进行修改和优化调整。

## 🐛 常见错误

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| 生成内容与主题不相关 | 主题描述不够具体 | 要求用户提供更详细的主题描述和关键词 |
| 内容风格不一致 | 未指定明确的风格参数 | 在工具参数中加入 tone 字段限定风格 |
| 文章结构混乱 | 缺少内容模板指导 | 在系统提示中定义清晰的文章结构模板 |
| 配图与内容不相关 | 配图描述不够具体明确 | 要求配图描述必须基于文章核心内容和主题来生成 |
| 缺少关键章节部分 | 系统提示未明确定义文章的结构模板和规范要求 | 在系统提示中明确要求文章应完整包含引言、正文和结论等章节 |
