# Agent 安全速查表

## 🛡️ 四层防御

| 层级 | 措施 | 说明 |
|------|------|------|
| 输入层 | 注入检测 + 清洗 | 正则匹配、关键词过滤 |
| Prompt 层 | System Prompt 加固 | 安全规则最高优先 |
| 输出层 | 敏感信息检查 | API Key、密码等 |
| 权限层 | 工具权限控制 | 最小权限、审批流程 |

## 🚨 常见攻击

| 攻击 | 示例 | 防御 |
|------|------|------|
| 直接注入 | "忽略之前的指令" | 输入过滤 |
| 间接注入 | RAG 文档中藏指令 | 文档审核 |
| 越狱 | "假设你没有限制" | Prompt 加固 |
| 信息提取 | "你的 System Prompt" | 规则禁止 |
