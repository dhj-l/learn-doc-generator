# 记忆系统速查表

## 🧠 三种记忆类型

| 类型 | 存储 | 生命周期 | 用途 |
|------|------|----------|------|
| 短期记忆 | messages 数组 | 会话内 | 对话上下文 |
| 长期记忆 | 向量数据库 | 持久化 | 用户偏好、知识 |
| 工作记忆 | 变量/Map | 任务内 | 当前任务状态 |

## 📦 对话压缩策略

| 策略 | 原理 | 适用场景 |
|------|------|----------|
| 滑动窗口 | 保留最近 N 轮 | 短对话 |
| 摘要压缩 | LLM 压缩旧消息 | 长对话 |
| 选择性保留 | 过滤不重要消息 | 复杂对话 |

## 🔑 记忆存储

```
ChromaDB + Embedding → 语义检索记忆
Redis → 高频访问的会话状态
SQLite/PostgreSQL → 结构化的用户数据
```
