# 第1章：AI-Native UI 模式

> 预计学习时间：60-80 分钟

## 💡 三种 AI-Native UI 模式

### 1. Chat Interface（对话界面）

最基础的 AI 交互方式——用户与 AI 对话。

```
┌──────────────────────────────────┐
│  💬 AI 助手                       │
├──────────────────────────────────┤
│  👤 用户: 帮我写一个登录组件       │
│                                  │
│  🤖 AI: 好的，这是一个 Vue 3 登录  │
│  组件...                          │
│  ```vue                         │
│  <template>...</template>       │
│  ```                            │
│                                  │
│  ┌──────────────────────┬────┐  │
│  │ 输入消息...           │ 发送│  │
│  └──────────────────────┴────┘  │
└──────────────────────────────────┘
```

### 2. Copilot（协作助手）

嵌入到工作流中的 AI，主动提供建议。

```
┌──────────────────────────────────┐
│  VS Code                         │
│  ┌─────────────────────────────┐ │
│  │ function login(user, pass) {│ │
│  │   // AI 建议 ↓              │ │
│  │   if (!user || !pass) {     │ │
│  │     return { error: '...' } │ │ ← AI 自动补全
│  │   }                         │ │
│  │ }                           │ │
│  └─────────────────────────────┘ │
└──────────────────────────────────┘
```

### 3. Agent Dashboard（Agent 控制台）

展示 Agent 的推理过程和工具使用。

```
┌──────────────────────────────────┐
│  🤖 Agent 执行日志                │
├──────────────────────────────────┤
│  ✅ 分析用户需求                  │
│  🔧 调用搜索工具 → 获取最新文档    │
│  📝 生成代码方案                  │
│  ⏳ 等待用户确认...               │
├──────────────────────────────────┤
│  [确认执行]  [修改方案]  [取消]    │
└──────────────────────────────────┘
```

---

## 📝 本章小结

- ✅ **Chat Interface** — 对话式交互
- ✅ **Copilot** — 嵌入工作流的辅助建议
- ✅ **Agent Dashboard** — 展示 Agent 执行过程
